Training Pipeline

How It Was Built

01
Dataset
15,000 images, 30 classes from Google Drive. Each class has default + real_world subfolders.
Google Drive · 30 classes
02
Preprocessing
Resize to 128×128, rescale to [0,1]. 80/20 train-val split via ImageDataGenerator.
ImageDataGenerator
03
Model
MobileNetV2 (ImageNet) frozen base. Custom head: GAP → Dense(128,relu) → Softmax(30).
TF · Keras · T4 GPU
04
Training
Adam optimizer, categorical crossentropy loss. 3 epochs, batch 16. Val acc: 82.40%.
3 epochs · batch 16
05
Deploy
Saved as .h5. Flask API serves predictions. This page calls /predict endpoint.
Flask · .h5 model
